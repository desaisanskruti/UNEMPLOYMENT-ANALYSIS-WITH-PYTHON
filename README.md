This project performs a detailed data analysis on unemployment trends using Python.
It explores unemployment rates across regions, time periods, and demographics, and visualizes key insights using charts.

. Features

 Trend analysis of unemployment over time

 Regional unemployment comparison

Visualization of unemployment patterns (bar chart, line chart, heatmap, etc.)

Complete data cleaning & preprocessing


 Technologies Used

.Python

pandas

numpy

matplotlib / seaborn / plotly
